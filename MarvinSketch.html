4. Markush Enumeration

Markush structures help you quickly create derivative structures by defining attachment groups.

Let’s create simple halogen derivatives of benzene.

  • Draw benzene ring and attach two carbons to create 1, 2 - xylene structure
  • Convert each of the attached methyl groups to R1 and R2 using right click options
  • To create double halogen derivatives, draw individual halogens on canvas, select them all and press R1 on keyboard to assign group R1 to them. Right click on each of the halogen atoms and select R-group attachment option. Repeat the step with R2
  • Go to structure - > Markush Enumeration

Cox-2 as a Target for novel specific inhibitor design

 

We will be using COX-2 and marketed drugs against COX as our toy system to understand various stages in computer aided drug design.

Theory behind the project: Cox-2 enzyme has been long known to be involved in prostaglandin production, which in turn translate the signal of pain. Common drugs from NSAIDs family target this system But there are many side effects, especially in chronic cases, which are believed to be due to cross reactivity of these molecules with Cox-1, an isoform of Cox. Let’s use the information at hand from various literature and databases about inhibitors for Cox and try to build optimized molecules which are specific only for Cox-2.

 

Compound Library Generation

Initial set of compounds in the library can come from various sources. We will use some basic sources like databases (PubChem), Designs (Molecules drawn or edited), Fragments (From other molecules) or Combinatorial.

 

a.         Use MarvinSketch software to draw structures of aspirin, Ibuprofen and celecoxib

 Save these three molecules with their respective names in MDL SDFile (*.sdf) in separate files.

b.         Use website PubChem (https://pubchem.ncbi.nlm.nih.gov/) to download structures of Valdecoxib, Rofecoxib, Nabumetone. These will be downloaded in SDF format.

c. Now that we have a small set of chemicals, let’s search for some more cox-inhibitors which are already known.

  • In order to discover the required molecules, a part of our library will be generated from the known compounds present in public databases.
  • Searching the PubChem may not give us all the results, so let’s search by the classes of chemical compounds. Remember, our target Cox-2 is an enzyme belonging to oxidoreductase class of enzymes.
  • Go to PubChem database, click on the Browse Data.
  • Select the classification level as chEMBL: Target tree, instead of MeSH
  • Search for compounds belonging to the Oxidoreductase as target class in the classification tree.
  • How many compounds are there in this class? Click on the number to open them in Entrez.
  • Filter the compounds based on known pharmacological action as Anti-Inflammatory Agents, Non-Steroidal.
  • Filter further with cyclooxygenase inhibitors.
  • Download all these compounds in SDF format, and save the file as coxibs.sdf for further use.

d. You have been provided here with various fragments from known molecules to be used in compound library generation. Use these fragments and build a combinatorial compound library using SmiLib tool. SmiLib required data in SMILES format. Use a simple MarvinSketch tool - Edit -> Copy as SMILES for molecules which are not in smiles format

Combining all sub-libraries with different format into a single library

Now that we have small libraries from various sources, let’s combine them into a single library. 3 molecules sketched from MarvinSketch, 3 molecules downloaded from PubChem and 24 molecules created using small combinatorial library in SmiLib, we should have a single file containing all 30 molecules. Follow these steps for combining

  • Open MarvinSketch
  • Open file small_library.sdf saved in earlier step
  • Go to file -> insert from file -> select the molecules saved earlier and downloaded from PubChem one by one. Repeat same for the file coxibs.sdf
  • Go to file -> save as . Save the library as combined.sdf. Do not forget to click on Save separate molecules.

No we should have a combined library which you can check by opening in Notepad and counting the number of lines.
